☁️ Sincronização Automática

☁️ Ativo

Dados sincronizados automaticamente via Firebase em tempo real. Funciona em qualquer dispositivo — basta fazer login.

✅ Auto-sync: cada alteração é enviada para a nuvem em 2 segundos.
Novo dispositivo / navegador: faça login — os dados aparecem automaticamente.
Offline: os dados ficam salvos localmente e sincronizam quando a conexão voltar.

💾 Backup Local — Exportar / Importar

Todos os dados (investigações, suspeitos, sites, licenças) são salvos automaticamente neste navegador. Use o backup para transferir entre dispositivos ou garantir uma cópia de segurança.

✅ Salvamento automático: cada alteração é gravada instantaneamente no localStorage deste navegador.
O backup .json permite restaurar os dados em outro dispositivo ou navegador.
